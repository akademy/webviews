WebViews

An example use of WebViews. iframes of various webpages. These all exist on the same domain as this page (which means I can access the content of the iframe).

The border of each view shows if it loaded correctly. There should be three green and one red on this page.

  • Green means load and check OK.
  • Yellow means something loaded correctly but I can't access the iFrame due to security restrictions.
  • Red means there was some error loading or that a check failed.
  • Blue is used when it is loading. (At the moment websites which timeout are also shown in blue)
